Forsaken Tool is a terminal-based, advanced \*\*OSINT (Open Source Intelligence) and Network Analysis\*\* framework. Organized with a modular architecture, it allows operators to gather intelligence swiftly and effectively on target infrastructures or entities.



\## 🚀 Features



\- 🌍 \*\*OSINT Core:\*\* IP Geolocation Lookup, Phone Number Recon, Global Username Scanner, EXIF Metadata Image Geo-Locator.

\- 🌐 \*\*Network \& Infra:\*\* Subdomain Mapper, DNS Zone Resolver, HTTP Header Grabber, Stealth Port Scanner.

\- 💻 \*\*Hardware Recon:\*\* MAC Address Vendor Lookup.

\- 📝 \*\*Automated Logging:\*\* Session outputs are automatically written to `forsaken\_report.txt`.
